O Círculo Cromático

Desenvolvedor: Matheus Oliveira da Silva Santos

Harmonia Utilizada: Monocromático

Cores Usadas: #0A0A41, #9595E5.

O Círculo Cromático é uma representação em círculo das cores percebidas pelo olho humano. Ele é composto por 12 cores: as três primárias, as três secundárias e as seis terciárias. Este instrumento um guia rápido de identificação das cores complementares, das análogas, das meio-complementares, e de outras combinações harmónicas possíveis.

Harmonias do Círculo Cromático

Monocromática, Análoga, Complementar, Triádica, Complemento Dividido, Dupla Complementar.

Monocromática

É a harmonia resultante de uma mesma cor da roda das cores.Utiliza variações de luminosidade e saturação de uma mesma cor.

Análoga

É a harmonia formada de uma cor primária combinada com duas cores vizinhas na roda das cores. Uma cor é utilizada como a dominante enquanto que as adjacentes são utilizadas para enriquecer a harmonia.

Complementar

É a harmonia que ocorre quando combinamos cores opostas na roda das cores. Uma harmonia complementar é intrinsecamente uma harmonia de contraste.

Triádica

É a harmonia onde usamos três cores equidistantes no circulo cromático. Esse tipo de combinação consegue dar um efeito visual muito atraente.

Complemento Dividido

É a harmonia onde usamos três cores equidistantes no circulo cromático. Esta harmonia não é tão contrastante como o esquema de complementares, mas aparece mais balançado e harmonioso

Dupla Complementar

Como o nome indica, refere-se a harmonia conseguida por dois pares de cores complementares entre si.

Ferramnetas para encontrar harmonias

Color Hunt, Color Lisa, Blend, Paletton, Adobe Color e Colordot.
